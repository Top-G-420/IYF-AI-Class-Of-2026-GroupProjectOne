# Group Project One - School Management System

## Description
This repository contains the Python program for our Group Project One: a **School Management System**. The system allows a school to manage students, teachers, classes, and grades efficiently. It is designed for educational purposes and can be extended to include more features such as attendance tracking, timetable management, and reports generation.

## Features
- Add, view, update, and delete student information
- Add, view, update, and delete teacher information
- Manage classes and assign students to classes
- Record and view student grades
- Simple command-line interface for easy interaction
